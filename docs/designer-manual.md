# GameInsight 设计师使用手册

## 欢迎使用 GameInsight

GameInsight 是一款专为游戏策划和设计师打造的 AI 驱动反馈分析工具。它能自动帮你收集、过滤、分析来自各大社交平台的玩家声音，让你从海量信息中快速找到真正有价值的改进方向。

---

## 1. 快速上手

### 1.1 配置你的项目
首次登录后，进入「设置」页面，输入：
- **游戏名称**：如 "英雄联盟"
- **监测关键词**：如 "LOL、英雄联盟、拳头"
- **目标平台**：勾选你要监测的渠道（微博、B站、TapTap 等）

配置完成后，系统会在每天自动开始采集和分析。

### 1.2 查看周报
每周一早晨，系统会自动生成上一周的反馈周报。你可以在首页看到：
- 本周一共收集了多少条玩家反馈
- 其中有多少条被 AI 判定为「有效诉求」
- 热门分类 TOP 3（如 UI/UX、数值平衡、BUG）
- 热点诉求卡片列表

---

## 2. 周报卡片详解

每张卡片代表一条 AI 提纯后的玩家诉求，卡片上会显示：

| 元素 | 说明 |
|------|------|
| 摘要 | AI 提炼后的核心诉求，1-2 句话 |
| 来源平台 | 微博 / B站 / TapTap / 小红书 / 抖音 |
| 玩家身份 | KOL / 从业者 / 资深玩家 / 新手玩家 |
| 情感分值 | -1（强烈负面）到 +1（正面），帮助你快速判断情绪烈度 |

### 2.1 查看详情
点击任意卡片，会平滑展开一个详情弹窗，里面包含：
- **原始内容**：玩家的完整发言
- **AI 分析摘要**：为什么这条反馈值得注意
- **来源链接**：点击可直接跳转原帖
- **相似诉求**：系统自动聚合的其他类似反馈

---

## 3. 从反馈到行动：待办管理

### 3.1 将诉求转为待办
在详情页中点击「转为待办」按钮，即可把这条诉求加入你的迭代看板。你可以：
- 添加自己的备注
- 设置优先级（高 / 中 / 低）

### 3.2 看板操作
进入「待办」页面，你会看到三列看板：
- **待处理**：刚加入的诉求，等待评估
- **设计中**：已经确认要改，正在设计或开发中
- **已完成**：已经上线或解决

点击卡片即可变更状态。

### 3.3 自动生成更新日志
当一条待办被标记为「已完成」时，系统会根据该诉求的摘要自动生成一段更新日志草稿。你可以在「发布素材」页面批量查看和复制这些文案，快速填充到游戏公告或社区运营内容中。

---

## 4. 导出与分享

### 4.1 导出周报
在周报页面点击「导出」按钮，可选择：
- **Markdown**：方便发在内部 Wiki 或 GitHub
- **PDF**：方便作为周报附件发送给领导或合作部门

### 4.2 分享链接
你可以把 GameInsight 的 Demo 链接直接发给 HR 或面试官，对方无需安装任何软件即可在线查看效果。

## 5. 常见问题

**Q: 为什么有些平台没有数据？**
A: 可能是因为该平台上关于你配置关键词的讨论较少，或者爬虫正在排队采集中。系统采用限速策略，避免被封禁。

**Q: AI 分析结果可以调整吗？**
A: 目前 AI 分析是自动的，但你可以在「详情页」看到原始内容并自行判断。后续版本会加入「标记分析错误」功能来持续优化模型。

**Q: 数据来源是否合规？**
A: GameInsight 仅采集公开可见的内容，不登录获取隐私数据。所有存储的用户 ID 都经过匿名化处理。如有玩家提出删除请求，你也可以在后台一键删除对应数据。
